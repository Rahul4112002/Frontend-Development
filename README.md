# Frontend Development

A comprehensive collection of frontend development learning materials covering HTML, CSS, and JavaScript.

## üìö Repository Structure

### 01. HTML
Complete HTML fundamentals and advanced concepts organized in levels:
- **Level 1**: Basic HTML structure, comments
- **Level 2**: Core elements (headings, paragraphs, images, videos, links, text formatting)
- **Level 3**: Semantic HTML (header, main, section, article, aside, footer, nav, div, span)
- **Level 4**: Lists, tables, iframes, and forms

### 02. CSS
CSS styling from basics to advanced layout techniques:
- **Level 1**: Selectors (inline, element, universal, id, class, group, descendant)
- **Level 2**: Colors (RGB, hex, alpha), background properties, visibility
- **Level 3**: Text styling (alignment, decoration, transform, fonts, icons)
- **Level 4**: Box model (padding, margin, border, border-radius, box-sizing)
- **Level 5**: Display properties, units (%, em, rem, vh, vw), positioning, z-index
- **Level 6**: Layouts (float, flexbox with all properties)

### 03. JavaScript
Complete JavaScript course materials from beginner to advanced:
- **Chapter 3**: HTML, CSS & JavaScript integration
- **Chapter 4**: Variables and data types
- **Chapter 5**: if-else, boolean, logical operators, scope, truthy/falsy
- **Chapter 6**: Functions
- **Chapter 7**: Objects
- **Chapter 8**: JSON, Local Storage, Date, and DOM
- **Chapter 9**: Arrays and loops
- **Chapter 10**: Advanced functions
- **Chapter 11**: Myntra Functional Clone template
